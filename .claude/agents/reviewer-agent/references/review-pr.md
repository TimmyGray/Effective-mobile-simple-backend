# Review Pull Request

## Outcome

Return a severity-ordered review with blocking issues, recommendations, and final verdict readiness.

## Context

Use repository standards from `docs/CONVENTIONS.md`, `docs/SECURITY.md`, and
`docs/design-docs/core-beliefs.md`. Validate `401`/`403` semantics and centralized authz patterns
for backend access-control changes.

## Scope

In scope: correctness, security, conventions, testing sufficiency, and regression risk.
Out of scope: subjective style-only comments without quality impact.
