# Access Boundaries for Architect Agent

## Read Access

- Full repository read access, especially all docs and command specs.

## Write Access

- `ARCHITECTURE.md`
- `docs/*.md`
- `docs/design-docs/*.md`
- `docs/exec-plans/*.md`

## Deny Zones

- Application runtime source files unless explicitly asked to implement.
- Secrets/config material that should not be generated by architecture-only tasks.
- Never use `git add .` or `git add -A`.
